# Instagram UTM Models

## Bio do Instagram

```
utm_source=instagram
utm_medium=social
utm_term=link_bio
```

**Exemplo de URL:**
```
https://exemplo.com/landing?utm_source=instagram&utm_medium=social&utm_campaign=L30&utm_content=organico&utm_term=link_bio
```

## Stories do Instagram

utm_source=instagram

**Exemplo de URL:**

https://exemplo.com/landing?utm_source=instagram&utm_medium=social&utm_campaign=L30&utm_content=organico&utm_term=link_stories

## Posts/Feed

```
utm_source=instagram
utm_medium=social
utm_term=post_feed
```

## Variações de utm_term

- `instabio` - Link na bio
- `instastories` - Stories
- `post_feed` - Posts no feed
- `reels` - Reels
- `igtv` - IGTV

## Exemplo Completo - Black Friday

```
https://exemplo.com/bf2024?utm_source=instagram&utm_medium=social&utm_campaign=BF24&utm_content=organico&utm_term=instastories
```
